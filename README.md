# DT-Frontend-Task1 — Professional Minimal UI

This project is a submission for the DeepTech Frontend internship Task 1.
It includes a minimal, professional UI implemented with plain HTML, CSS, and JavaScript.

## Files
- `index.html` — main page
- `style.css` — styles (responsive, centered header)
- `script.js` — dynamic JSON loading and small interactions

## How to preview
Open `index.html` in a browser (double-click or right-click → Open with).  
Click **Load Sample JSON** to populate skills, and **Submission Checklist** to open modal.
